Ин комада айни вазифаи `/stop`-ро иҷро мекунад. Файли `.claude/commands/stop.md`-ро пайравӣ кун:

1. PROGRESS.md-ро бо корҳои ин session нав кун (тамомшуда + ҳолат + қадами навбатӣ + сана).
2. Агар `origin` пайваст набошад — GitHub repository бисоз ва пайваст кун.
3. `git add -A`, `git commit`, `git push -u origin main`.
4. Ба корбар ҳисобот деҳ.
